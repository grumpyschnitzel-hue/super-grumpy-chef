# 🍽️ Super Grumpy Chef — Restaurant Website

A showcase restaurant website built with Claude Code.

## Project Overview

- **Brand:** Super Grumpy Chef
- **Type:** Restaurant / hospitality website
- **Built with:** HTML, CSS, Vanilla JS (no frameworks required)
- **Claude Code Project:** Yes — see `CLAUDE.md` for AI instructions

## Project Structure

```
super-grumpy-chef/
├── index.html          # Main entry point
├── style.css           # Design tokens + global styles
├── main.js             # App logic (menu filter, reservations, etc.)
├── assets/
│   ├── images/         # Hero, food photography, team
│   └── icons/          # SVG icons / logo
├── CLAUDE.md           # Claude Code instructions & context
└── README.md
```

## Pages / Sections

- [ ] Hero (brand statement + CTA)
- [ ] About (story of the chef)
- [ ] Menu (filterable dishes)
- [ ] Gallery
- [ ] Reservations (form)
- [ ] Contact / Location

## Getting Started

```bash
git clone https://github.com/grumpyschnitzel-hue/super-grumpy-chef.git
cd super-grumpy-chef
# open index.html in browser OR use a local dev server:
npx serve .
```
